大风降温影响北方 东北局地有大雪

2020-12-23 07:49:33 来源： 中国天气网
分享到： 原创未经授权不得转载

中国天气网讯 预计，今明两天（12月23日至24日），冷空气主要影响北方，带来大风降温天气，局地降温幅度将超过8℃。降水方面，今明天全国雨雪先增后减，东北局地今有大雪。

北方大风降温进行时 局地降幅将超8℃

昨天，全国大部维持较常年偏暖的格局。不过，受冷空气影响，内蒙古西部和甘肃西部等地出现4-10℃降温。

今明两天，冷空气继续东移，主要影响北方地区，部分地区将有大风降温天气。中央气象台预计，华北中北部和东北地区等地有4～6级偏北风，阵风可达8～9级；华北、东北地区、黄淮等地气温下降4～6℃，局地降温幅度超过8℃。

其中，东北东部降温最明显，城市中，哈尔滨、沈阳、长春等地明天单日降温幅度都在6℃左右，公众请注意及时添衣保暖。

展望中后期，预计，25日至26日，大部地区气温回升。27日以后，又有较强冷空气影响中东部地区，公众请注意关注临近预报信息。

贵阳近日迎来久违的冬日暖阳。（图/石开银）

降水方面，昨天，全国雨雪仍然较少，内蒙古东北部出现小雪。

今天，东北等地降雪范围较大，南方阴雨主要出现在华南大部、江南东南部等地。预计，新疆阿勒泰山区、内蒙古东部、黑龙江大部、吉林东部等地的部分地区有小到中雪，其中，黑龙江东部局地有大雪。江南东南部、华南大部、贵州西部、云南东部、台湾岛等地的部分地区有小雨，局地中雨。

明天，全国雨雪范围又会缩减，预计，内蒙古东北部、黑龙江西北部和东南部、吉林东部、西藏东部、青海南部、川西高原北部等地部分地区有小到中雪或雨夹雪，其中，西藏东部局地有大雪。四川盆地东部和南部、重庆大部、贵州中西部、云南东部、台湾岛大部等地部分地区有小雨。

气象专家提醒，今天东北等地部分地区将有降雪，公众雪天出行请留意交通安全。西南部分地区今明天多阴雨，当地公众请注意防范。
